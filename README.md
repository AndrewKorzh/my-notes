# my-notes

## Идея:
Небольшое веб приложение с микросервисной архитектурой для хранения записей различных пользователей

Бэк - python + FastAPI

Фронт - `react js`, а может просто `html + js`

Базовые настройки портов (временно): `api - 5000`, `auth - 8000`, `notes - 8080`
